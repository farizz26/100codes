# 100 Codes: Daily C Coding

Welcome to **100 Codes: Daily C Coding**!  
This project is a personal journey by (@farizz26) to practice and improve my C programming skills by solving coding challenges every day.

## Project Overview

- **Goal:** Solve 100 coding problems in C, one per day.
- **Language:** C (C Language)
- **Editor:** Visual Studio Code (VS Code)
- **Author:** Fariz

## Getting Started

### Prerequisites

- [GCC](https://gcc.gnu.org/) or any C compiler
- [Visual Studio Code](https://code.visualstudio.com/)
- (Optional) C/C++ VS Code extension for better development experience

### Running the Code

1. Clone the repository:
    ```sh
    git clone https://github.com/farizz26/100-codes.git
    cd 100-codes
    ```
2. Open the folder in VS Code.
3. Compile any C file you want to try:
    ```sh
    gcc filename.c -o filename
    ./filename
    ```

## Folder Structure

- Each code/problem will be in its separate `.c` file.
- Filenames are typically numbered or named according to the problem.
